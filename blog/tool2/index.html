🚀 Tool Gửi Tin Nhắn Zalo Tự Động

Kết nối Google Sheets ➝ Lọc học viên chưa đóng học phí ➝ Gửi tin nhắn Zalo tự động

📖 Giới thiệu

Đây là một công cụ được viết bằng C#, sử dụng Google Sheets API và Selenium WebDriver để tự động gửi tin nhắn Zalo tới phụ huynh học viên.

  • ✅ Đọc dữ liệu từ Google Sheets.
  • ✅ Lọc học viên chưa đóng học phí.
  • ✅ Tự động mở Zalo Web và gửi tin nhắn.
  • ✅ Cá nhân hóa nội dung: tên học viên, lớp, số tiền.

🛠 Công nghệ sử dụng

  • Google Sheets API ➝ Lấy dữ liệu học viên.
  • Selenium + ChromeDriver ➝ Điều khiển trình duyệt.
  • Zalo Web ➝ Nơi gửi tin nhắn.
  • C# .NET ➝ Ngôn ngữ lập trình.
  • OAuth2 Service Account ➝ Kết nối Google Sheets.
  • Console App ➝ Giao diện chạy tool.

📌 Quy trình hoạt động

  1. 📂 Đọc danh sách học viên từ Google Sheets.
  2. 🔍 Lọc ra học viên có trạng thái CHƯA ĐÓNG.
  3. 🌐 Mở Zalo Web với Chrome.
  4. 💬 Gửi tin nhắn nhắc nhở học phí đến phụ huynh.
